## Features

- **Live packet monitor** — RX/TX direction, timestamp, gap, and byte highlighting
- **Stream splitter** — Parse raw byte streams into packets via SOF/EOF delimiter, length field, gap-based, or regex
- **Checksum validation** — CRC8/16/32, Modbus, DNP3, Fletcher, XOR, and custom JS formula
- **Transmit** — HEX/ASCII quick-send, F1–F12 macros, trigger-based auto-send, script runner
- **Analyzer** — Gap timing stats, byte frequency, TX→RX cycle analysis
- **Auto-update** — Automatically detects new releases and installs in one click
